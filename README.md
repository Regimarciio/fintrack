# FinTrack

Sistema inteligente de leitura de documentos financeiros com IA.

## Funcionalidades
- Upload de faturas, notas e tickets
- OCR automático
- Extração de dados com IA
- Dashboard de gastos
- Classificação automática

## Tecnologias
- Python
- Flask
- OpenAI
- SQLite
- Streamlit

## Como rodar

pip install -r requirements.txt
python app.py
streamlit run dashboard.py
